## 📡 What is HACK KAP?

**HACK KAP** is a Sri Lanka–based cybersecurity platform founded in **2022**, built on a single belief:

> *The gap between knowing security and doing security is where most defenders fail — and where we train.*

We are not a course catalog. We are a **living cyber lab** — an evolving ecosystem where learners, researchers, and professionals sharpen their skills through realistic attack simulations, structured learning paths, and hands-on CTF challenges that mirror the complexity of real-world threats.

Whether you're cracking your first binary or contributing CVE research, HACK KAP is your arena.

## 🏆 CTF Challenges

HACK KAP hosts and maintains an active library of **Capture The Flag** challenges built to reflect genuine threat scenarios — not toy problems.

```
🟢 EASY       ████████████░░░░░░░░  Foundations, recon, basic web
🟡 MEDIUM     ████████████████░░░░  Real CVEs, logic flaws, crypto attacks  
🔴 HARD       ████████████████████  Zero-day style, chained exploits, kernel
```

**Challenge Design Philosophy:**
- 🎯 Every challenge is based on **real vulnerability classes**
- 🔗 Multi-step challenges that require **chaining techniques**
- 📝 Official writeups released post-CTF for community learning
- 🔄 Challenges updated regularly to reflect **current threat landscape**

## 📜 Code of Ethics

> **HACK KAP operates on a strict ethical foundation.**

```
✅  Practice only on platforms and systems you own or have explicit permission to test
✅  All challenges exist in isolated lab environments
✅  Responsible disclosure is mandatory for real-world vulnerability findings
✅  Community members are expected to uplift, not gatekeep
❌  Unauthorized access to third-party systems is strictly prohibited
❌  Using skills from this platform for malicious purposes violates our ToS
```

*With great skill comes greater responsibility.*
